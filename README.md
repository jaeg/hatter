# Wart Control

#### Commands
- proxy
  - provides endpoints for wart control UI
- apply <env file path>
  - applies environment file to cluster
- purge
  - destroys cluster

- threads
  - lists threads
- threads disable <name>
  - stops thread
- thread enable <name>
  - starts thread

- endpoints
  -lists endpoints
- endpoints disabled
  - disables endpoint
- endpoints enable
  - enables endpoint

- warts
  - lists warts
- warts stop <name>
  - stops wart
